# Button

This is the `RusticUI` Button Component. It supports all the HTML button attributes and some extra.

This is the button when it is loading. It replaces all children with a Loading Animation and the text **Please Wait**

### Disabled State

This is the button when it is disabled. It applies for all color variants and the color becomes faded when disabled
